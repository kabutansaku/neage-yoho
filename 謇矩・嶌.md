# 値上げ予報サイト 完全セットアップ手順書

**月額コスト：約¥50**（Claude API利用分のみ。他は完全無料）  
**所要時間：約30〜40分**（初回のみ）

---

## 全体の構成

```
あなたのPC（管理画面）
    ↓ admin.html でログイン
    ↓ 記事を書く or AIで生成
    ↓ articles-index.json をダウンロード
    ↓ GitHubにアップロード
GitHub Pages（公開サイト）← 一般ユーザーが見るサイト
    ↑
GitHub Actions が毎朝・毎週自動でデータを更新
```

---

## ファイル一覧

| ファイル | 役割 |
|---------|------|
| `index.html` | 公開トップページ |
| `admin.html` | 管理画面（パスワード保護） |
| `fx-data.json` | 為替レート（毎朝自動更新） |
| `articles-index.json` | 記事一覧（管理画面からエクスポート） |
| `articles/` | 記事HTMLが入るフォルダ |
| `generate_articles.py` | AI記事生成スクリプト |
| `.github/workflows/update-fx.yml` | 毎朝9時 為替更新 |
| `.github/workflows/generate-articles.yml` | 毎週月曜 AI記事生成 |

## STEP 1：GitHubアカウントを作る（5分）

1. https://github.com を開く
2. 「Sign up」をクリック
3. メールアドレス・パスワード・ユーザー名を入力
4. メール認証を完了する

---

## STEP 2：リポジトリを作る（3分）

1. GitHubにログイン後、右上「＋」→「New repository」
2. 以下のように設定する：
   - **Repository name**：`neage-yoho`
   - **Public**（必須 ※GitHub Pages 無料公開に必要）
   - 他はデフォルトのまま
3. 「Create repository」をクリック

---

## STEP 3：ファイルをアップロードする（5分）

1. 作成したリポジトリのページで「uploading an existing file」をクリック
2. ZIPを解凍したフォルダの**中身をすべてドラッグ＆ドロップ**
   - `index.html`
   - `admin.html`
   - `fx-data.json`
   - `articles-index.json`
   - `generate_articles.py`
   - `.github/` フォルダ（ワークフローが入っています）
   - `articles/` フォルダ
3. 「Commit changes」をクリック

> ⚠️ `.github/` フォルダが見えない場合：macOSでは「.（ドット）から始まるファイルを表示」を有効にしてください（Cmd + Shift + .）

---

## STEP 4：管理画面のパスワードを変更する（2分）

1. ダウンロードしたZIPを解凍する
2. `admin.html` をテキストエディタで開く
3. 以下の行を探す：
   ```javascript
   const DEF_PW = 'admin1234';
   ```
4. `'admin1234'` の部分を自分のパスワードに変更して保存
5. 変更した `admin.html` を再度GitHubにアップロード

---

## STEP 5：GitHub Pagesを有効にする（3分）

1. リポジトリの「Settings」タブを開く
2. 左メニューの「Pages」をクリック
3. **Source**：「Deploy from a branch」
4. **Branch**：`main`、フォルダ：`/ (root)`
5. 「Save」をクリック
6. 数分後に以下のURLでサイトが公開される：  
   `https://あなたのユーザーID.github.io/neage-yoho/`

---

## STEP 6：Claude APIキーを取得する（5分）

1. https://console.anthropic.com にアクセス
2. 「Sign up」でアカウント作成（メールアドレスのみ）
3. 「API Keys」→「Create Key」をクリック
4. 表示されたキー（`sk-ant-...`）をコピー・保存

> **料金目安**：1記事生成あたり約1〜3円。週3本×4週 = 月約50円。

---

## STEP 7：APIキーをGitHubに登録する（3分）

自動生成（GitHub Actions）で使うために設定します。

1. リポジトリの「Settings」タブ
2. 左メニュー「Secrets and variables」→「Actions」
3. 「New repository secret」をクリック
4. 以下を入力：
   - **Name**：`ANTHROPIC_API_KEY`
   - **Secret**：コピーしたAPIキー（`sk-ant-...`）
5. 「Add secret」をクリック

---

## STEP 8：動作確認（手動実行）（5分）

### 為替更新を手動実行する
1. リポジトリの「Actions」タブ
2. 左メニュー「為替レート 毎日更新」を選択
3. 「Run workflow」→「Run workflow」
4. 緑のチェック ✅ が付いたら成功
5. `fx-data.json` が更新されているか確認

### AI記事生成を手動実行する
1. 左メニュー「AI記事 週次自動生成」を選択
2. 「Run workflow」→「Run workflow」
3. 緑のチェック ✅ が付いたら成功
4. `articles/` フォルダに HTML ファイルが生成されているか確認
5. `articles-index.json` が更新されているか確認

## 管理画面の使い方

### アクセス方法
```
https://あなたのID.github.io/neage-yoho/admin.html
```
このURLは一般ユーザーには教えないでください。

### コラム記事を書いて公開する手順
1. `admin.html` にアクセス → パスワードを入力
2. 左メニュー「新規記事作成」をクリック
3. タイトル・説明文・本文を入力
4. 種類：「コラム（手動執筆）」を選択
5. 「✅ 公開する」をクリック
6. 自動的に「エクスポート」ページに移動する
7. 「articles-index.json をダウンロード」をクリック
8. GitHubリポジトリの**ルート**にアップロード
9. 数分後にトップページに表示される 🎉

### AIで記事を生成する手順
1. 「新規記事作成」→ 右側の「✨ AIで記事を生成する」
2. テーマを選択 → 「✨ AIで記事を生成する」をクリック
3. 生成された記事を確認・編集
4. 「✅ 公開する」→ エクスポート → GitHubにアップ

### AI一括生成（週次まとめて）
1. 左メニュー「AI一括生成」
2. APIキーを入力して「保存」
3. 対象月・為替レートを確認
4. 「✨ まとめて生成」をクリック
5. 3記事が下書き保存される
6. 「記事一覧」で確認 → 「公開」→「エクスポート」→ GitHubにアップ

## よくある質問

**Q：為替レートが「---」のまま表示される**  
A：GitHubの「Actions」→「為替レート毎日更新」を手動実行してください。

**Q：記事がトップページに表示されない**  
A：「エクスポート」から `articles-index.json` をダウンロードして、GitHubにアップしましたか？

**Q：AI生成でエラーが出る**  
A：「AI一括生成」ページでAPIキーが正しく入力されているか確認してください。

**Q：パスワードを忘れた**  
A：`admin.html` の `const DEF_PW = '...'` を変更してGitHubにアップしてください。

**Q：独自ドメインを使いたい**  
A：お名前.com などで年約1,000円のドメインを取得し、GitHubの Pages 設定でカスタムドメインを登録できます。
